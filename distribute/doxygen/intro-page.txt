<h2>Welcome to the Moai beta. &nbsp;</h2>
<h3>Thanks for installing the SDK. &nbsp;Here's the scoop:</h3>
<h4>Sample code:</h4>
Go to the samples folder, choose a sample, then execute run.bat to see it perform. &nbsp;One of our priorities is to build richer samples for Moai developers, so if you'd like to see a particular code sample, let us know by posting in the <a href="http://getmoai.com/forums/">Moai Forums</a>.<br>
<h4>FAQ:</h4>
<a href="http://getmoai.com/get-started/moai-faq/">The Moai FAQ</a> is available on getmoai.com.<br>
<h4>Moai Cloud:</h4>
You already have a Moai Cloud account and you can upload Lua scripts to be hosted and run as web services via the <a href="http://dashboard.moaicloud.com/">Moai Cloud Dashboard</a>. &nbsp;&nbsp;If you have support questions,
post them to the <a
href="http://getmoai.com/forums/moai-cloud-developer-support/">Moai
Cloud Developer Support Forum</a> or check out the <a
href="http://getmoai.com/wiki/">Moai Wiki</a>.<br>
<h4>Moai SDK Updates:</h4>
We'll email you about major new revisions during the beta period.
&nbsp;Stay tuned.<br>
