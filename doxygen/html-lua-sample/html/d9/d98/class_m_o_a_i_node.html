setAttrLink ( attributeID, sourceNode, sourceAttribute )

Sets an attribute of a Moai object.

Parameters:
attributeIDProperty of this object to link to.

  • ATTR_X_LOC - Location of the object along its X-axis.
  • ATTR_Y_LOC - Location of the object along its Y-axis.
  • ATTR_Z_ROT - Rotation of the object about its origin.
  • ATTR_X_SCL - Scaling of the object along its X-axis.
  • ATTR_Y_SCL - Scaling of the object along its Y-axis.
  • ATTR_BRUSH_ID - Index of a MOAIBrush in a MOAISpriteLibrary.
sourceNodeObject to link to.
sourceAttributeAttribute of the object you are linking to to connect to.

  • ATTR_TIME - The time of a MOAITimer object.
  • ATTR_VALUE - The value parameter of a MOAIAnimCurve key.
