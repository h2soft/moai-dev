setScale

setScale ( self, xScale, yScale )

Sets the number of world units visible of the viewport for one or both dimensions. Set '0' to use derived value.

Parameters:
self(in)
xScale(in) Total number of world units visible along width of viewport or 0.
yScale(in) Total number of world units visible along height of viewport or 0.
