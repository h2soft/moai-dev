load

load ( self, imagefile|data, charCodes )

Loads an image to use as a font.

Parameters:
self(in)
imagefileImagefile to be loaded. Must be a .bmp, .png, or .jpg.
dataA MOAIData object containing image data. Must be one of the filetypes above.
charCodesA string that sets the default character assignments of the font. For example, if A and B are the first letters in the font image, the first characters of the string would be "AB" and so forth.
loadFromTTF

() loadFromTTF ( self, filename, charcodes, points, dpi )

Attempts to load glyphs contained in a TTF into a texture for use as a bitmap font. Texture size is presently limited to 1024x1024. Unicode support coming Soon(tm).

Parameters:
self(in)
filename(in) Name of a .ttf font file.
charcodes(in) List of glyphs to be rendered to texture.
points(in) Point size to be rendered.
dpi(in) Device DPI (points per inch of device screen).
