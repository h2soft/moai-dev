getSize

getSize ( )

Returns the size of the string obtained from an httpPost or httpGet call.

Returns:
String size. If the call found nothing, this will return the value zero (not nil).

Returns:
The text string.
httpGet

httpGet ( url )

Sends an API call to the server for downloading data. The callback function (from setCallback) will run when the call is complete.

Parameters:
urlA string containing the full url of the API call.
httpPost

httpPost ( url, string|data )

Posts an API call to the server for uploading data. The callback function (from setCallback) will run when the call is complete.

Parameters:
urlA string containing the full url of the API call.
stringA character string containing text to send as POST data.
dataA MOAIData object to send as POST data.
parseXml

parseXml ( url )

Parses an XML table obtained from httpGet.

Returns:
Parsed XML code.
setCallback

setCallback ( callback )

Sets the function to be called when an httpGet or httpPost is completed.

Parameters:
callbackThe function to be called. The completed HTTPtask is passed into the function as the first argument.
