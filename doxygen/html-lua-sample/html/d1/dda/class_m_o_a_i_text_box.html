spool

( spoolAction ) spool ( self )

Sets text box reveal to 0 (i.e. hides all visible text) and returns a spool action that, when run, will increment the reveal count to animate the text appearing letter by letter. Action is automatically added to the root of the action tree, but may be reparented or stopped by user. See MOAITextSpoolAction for instructions on how to page text and throttle spool speed.

Parameters:
self(in)
