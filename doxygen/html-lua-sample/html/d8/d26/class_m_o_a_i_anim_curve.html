setKey

setKey ( self, index, time, value, modde )

Sets the location and properties of keyframes in the curve.

Parameters:
self(in)
index(in) The index of this keyframe in the table (starts at 0).
time(in) The timestamp of the keyframe, or when it occurs.
value(in) The value of the keyframe.
mode(in) The mode of the keyframe. Essentially the shape of the curve. (optional , default is SMOOTH)

  • EASEIN - Slows down as it approaches the next keyframe.
  • EASEOUT - Comes slowly out of the current keyframe.
  • FLAT - No curve. Keeps the same value until another keyframe is reached (best for animation frames).
  • LINEAR - A generic straight line that maintains a steady speed to the next keyframe.
  • SMOOTH - Comes slowly out of the current keyframe and slows into the next one.
  • SOFT_EASEIN - Like EASEIN, but with a less snappy curve.
  • SOFT_EASEOUT - Like EASEOUT, but with a less snappy curve.
  • SOFT_SMOOTH - Like SMOOTH, but with a less snappy curve.
