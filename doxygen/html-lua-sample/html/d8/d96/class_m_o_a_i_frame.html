setLayout

() setLayout ( self, xFit, yFit )

Sets the layout mode for each access. Can be MOAIFrame.LAYOUT_ALIGN_MIN, MOAIFrame.LAYOUT_ALIGN_MAX or MOAIFrame.LAYOUT_ALIGN_CENTER, MOAIFrame.LAYOUT_JUSTIFY_MIN, MOAIFrame.LAYOUT_JUSTIFY_MAX or MOAIFrame.LAYOUT_JUSTIFY_CENTER. In this case, MIN, CENTER and MAX refer to the border along each axis. So on the x axis, MIN would be left and MAX would be right.

Parameters:
self(in)
y(in)

setWeight

() setWeight ( self, xWeight, yWeight )

Sets the stretch weight for stretchable layours (i.e. FIT_EXPAND). This is the proportional stretch relative to all other stretchable layouts. Factors are evaluated relative to each other, so any number space can be used: 1, 2, 5 is equivalent to 0.5, 1, 2.5.
